Hello World: The Front

Turns out that design is pretty hard - who knew?

Web Development Sass

After building a static site generator and writing some content, all I had left to do was make it look pretty. How hard could that possibly be?

Turns out it was a lot harder than I expected.

Sass

CSS has a reputation for being pretty painful. In recent times, it's gotten slightly better with things like flexbox. One of these days I'll take some time and learn it properly.

For the styling of this website, I'm using Sass. Sass doesn't magically help me master flexbox, but it definitely makes life easier. Being able to nest selectors logically helps keep everything a lot more readable. I can declare all my colours in a shared file, and try out different accent colours with just one line changed.

I'm not doing anything too fancy style wise, so this part went relatively smoothly. It's hard to go wrong with a single column blog layout.

Colo(u)rs

Surprisingly, my favourite colour is orange - but orange is a controversial colour, so I decided it's probably for the best not to use that on my personal website. Usually, I'm a sucker for a light blue and white colour scheme.

I wanted to colour the links in the text to add a splash of colour to otherwise drab articles, but unfortunately the contrast between light blue and white doesn't work all that well. I'm not sure how important or accurate this contrast rating really is; but I'd rather not get into a fight over that so I looked for alternative options instead. I ended up settling on a nice shade of purple - it'll never replace light blue in my heart, but at least the contrast score is good.

If you're reading this in the future, there's a good chance I've changed up the links or adjusted the accent colour already. Sorry.

Lighthouse

Lighthouse is an automated tool for auditing webpages. It's great for catching pesky performance problems, or picking up on accessibility issues. Since this is a pretty lightweight blogging site, I want my performance to be as good as possible.

Overall, nothing was too badly wrong but there were a couple of things I had to do:

  • Make sure I had alt text and aria-labels on everything that needed it
  • Defer some scripts (such as Font Awesome) and preload some resources (such as fonts)
  • Add some more boring meta tags

Once that was all tweaked, I ended up with a perfect score!

I'm sure this won't last all that long, but it's great to start off on the right foot. Lighthouse also provides some Continuous Integration tools - in future, I'd like to set these up to automatically run on the site using Github Actions.

Open Graph Tags

Open Graph Tags are often overlooked, but they're surprisingly useful to have. These tags are used by most social media services to generate the fancy embed snippets that appear when links are shared.
